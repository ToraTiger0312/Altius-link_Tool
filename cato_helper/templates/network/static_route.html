Static Route追加（Site）

このツールについて
Static Route 追加時に参照したい Site / Network / SDP IP Range の一覧を取得します。

この画面では、CMA にログイン済みの環境に対して GraphQL を実行し、 以下の情報をまとめて取得して表示します。

  • Site 一覧（Site 名 / Site ID）
  • 各 Site の Network 一覧（Interface 名 / Type / Subnet / Gateway など）
  • SDP リモートユーザーの IP Range（Default / Dynamic / Static）

CMA にログインしていない場合は、まず右上の「CMA ログイン」からログインしてください。
